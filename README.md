# Arbol-de-Max-Min-Kruskal-P5
El algoritmo utiliza la clase DisjointSet. Al ordenar las aristas, si eliges el modo "MIN", el algoritmo toma siempre la arista más barata disponible. Si cambias el parámetro a "MAX", la función sorted() invierte el orden (reverse=True), obligando a Kruskal a comportarse como un buscador del árbol de máximo coste.
